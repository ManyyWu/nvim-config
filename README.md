# nvim-config

## 介绍
  Neovim纯Lua配置
  
## 环境
  1. Neovim >= 0.7: 必须
  2. ripgrep: 必须
  3. sed: 必须
  4. gnu-sed: macOS必须

## 安装
  ```Shell
  git clone https://github.com/ManyyWu/nvim-config.git && cd nvim-config && ./install.sh
  ```

## 卸载
  ```Shell
  ./uninstall.sh
  ```
